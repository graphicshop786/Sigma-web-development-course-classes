Style Types In CSS

  1. Inline Style
  2. Internal Style
  3. External

Inline Style

Inline styles are applied directly to the HTML elements using the `style` attribute.

Code used:

<p style="color: tomato; background-color: black; padding: 15px; font-size: 18px; border-radius: 5px;">...</p>

Internal Style

Internal styles are defined within the <style> tag inside the <head> section of the HTML document.

Code used:

<style>
#internal p {
color: white;
background-color: navy;
padding: 15px;
font-size: 18px;
border-radius: 5px;
}
</style>

External Style

External styles are defined in an external CSS file which is linked to the HTML document using the <link> tag.

Code used:

/* In styles.css */
#external p {
color: green;
background-color: lightyellow;
padding: 15px;
font-size: 18px;
border: 2px dashed black;
border-radius: 5px;
}
/* In HTML */
<link rel="stylesheet" href="styles.css">

By using these different types of CSS adding, you can style your HTML documents in various ways to create visually appealing web pages.
