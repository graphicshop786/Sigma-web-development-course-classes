CSS Cascade aur Specificity Samajhne Ke Examples

Example 1: Simple Element Selector

p { color: gray; }
/* Commented Out: Agar ye hota to gray ki jagah black hota */
/* p { color: black; } */

Output: Ye gray color mein hai.

Specificity: 1 (sirf element selector)

Example 2: Class Selector

.example1 { color: blue; }
/* Commented Out: Agar ye active hota to blue ki jagah pink hota */
/* .example1 { color: pink; } */

Output: Ye blue color mein hai.

Specificity: 10 (class selector)

Example 3: Multiple Classes

.example2 { color: red; font-weight: bold; }
/* Commented Out: Ye rule agar chalta to red ki jagah brown hota */
/* .example2 { color: brown; } */

Output: Ye red aur bold hai.

Specificity: 10 (aik class selector)

Example 4: Element + Class

p.example3 { color: green; }
/* Commented Out: Ye simple p ko target karta, specificity kam hoti */
/* p { color: teal; } */

Output: Ye green color mein hai.

Specificity: 11 (element = 1 + class = 10)

Example 5: Nested Selector

.example4.highlight { background-color: yellow; }
/* Commented Out: Agar ye hota to sirf class pe gray background lagta */
/* .example4 { background-color: gray; } */

Output: Ye yellow background wala hai.

Specificity: 20 (do class selectors = 10 + 10)

Example 6: ID Selector

#example5 { color: purple; }
/* Commented Out: Ye rule agar hota to purple ki jagah maroon hota */
/* #example5 { color: maroon; } */

Output: Ye purple color mein hai.

Specificity: 100 (ID selector)

Example 7: Parent aur Child Selector

.example6 p { color: orange; }
/* Commented Out: Ye simple p ko target karta, lekin specificity kam hoti */
/* p { color: cyan; } */

Output: Ye orange color mein hai.

Specificity: 11 (class = 10 + element = 1)

Example 8: Cascade Order Check

.example7 { color: black; }
.example7 { font-style: italic; }
/* Commented Out: Agar ye baad mein hota to italic ke sath color bhi change hota */
/* .example7 { color: violet; } */

Output: Ye italic hai, lekin color gray hai kyunki p ka rule overwrite karta hai.

Specificity: 10 (class selector), lekin element rule (p) cascade mein priority leta hai.
